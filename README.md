# ASRCTF 2026 Official Repository

This repository contains qualifiers and finals challenges for ASRCTF 2026.

## Packaging Rules
- **Solve Scripts**: Real solve scripts belong under `sol/solve.py`. Do not include solve scripts in the public `dist/` folders.
- **Starter Harnesses**: Starter templates/harnesses may be included in the public `dist/` folder for competitors, but must not contain the solution.
